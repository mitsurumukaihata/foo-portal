🚗 車番不一致明細 (弥生備考に別車番のヒントあり)

売上明細の車番と弥生備考の文字列が違うケース。弥生入力時に複数車両を1伝票で処理した可能性。

🔁 重複車両ペア (顧客別)

同じ顧客内で「短い車番が長い車番のサフィックスと一致」する重複候補を検出。 正しい方にチェック → 統合実行で売上明細を移管し、もう一方を削除します。

読込中...

🔧 サカイ系車両を支店別に振分

振替先は サカイ広島支社 / サカイ広島北 / サカイ広島南 / サカイ広島西 の4択のみ。
行をチェック → プルダウンで支店選択 → 一括振替で実行。

読込中...

🆔 番号不明リスト (車番なし or 管理番号なし)

事務員さんが手入力で補完するためのリスト。タブで切替:
・車番なし: 「管理#XXX」「数字のみ」など、フル車番が未入力
・管理番号なし: 管理番号運用顧客 (リョーキ・ムロオ等) で 管理番号空欄
🔢 車番不明
🆔 管理番号不明

読込中...

👤 顧客振分タスク (車両を別顧客に振替)

① 元顧客を選択 → 配下の車両一覧表示
② 各車両の振替先をプルダウンで選択 (チェック)
③ 「選択行を一括振替」で実行
例: サカイ広島西の78台 → 北/南/西に振り分け

🛠 一括クリーンアップ

サービス系明細(脱着/バランス/Fバランス/廃タイヤ)のタイヤサイズはテンプレ残りで意味なしのケースが多いため、一括NULLクリアできます。
